Macana は Obsidian Vault を基に静的な HTML サイトを生成するツールです。

![Logo|64](../Assets/logo.svg)

## 機能

### 重視するもの

- 静的な HTML を出力する
- JS がなくても最低限の閲覧が行えるマークアップ
- 最小限の設定でいい感じにシンプルな出力

### 重視しないもの

- Obsidian 以外のサポート
	- 基本はファイルベースの Markdown なのである程度は動くが、 Obsidian Vault の対応が最優先
- カスタマイズ
